# MEDIAPly Preview
1. 플레이 리스트 영상 편집을 편리하게 만들 수 있는 웹기반 플레이 리스트 편집기 입니다. 
2. 로컬 PC에서 MEDIAPly Preview.html 파일을 더블 클릭 하는 것만으로 툴을 실행 할 수 있습니다.
3. 윈도우, 맥, 스마트패드 을 가리지 않고 브라우저가 있는 장비들에서 구동 시킬 수 있습니다.
4. UI는 로컬 PC를 기준으로 제작하였기에 스마트 기기에서는 편집제어가 용이하지 않을 수 있습니다.

## 미디어플리(프리뷰) 기능 설명
1. 타임라인에 총 5개까지 트랙을 추가 할 수 있습니다.
2. 각 트랙별로 바탕 이미지 또는 동영상을 불러 올 수 있으며, 오디오 파일 역시 불러 올 수 있습니다.
3. 트랙간의 전환효과를 사용 할 수 있습니다.
4. 몇가지 이펙트 효과를 실정하고 있으므로 풀다운 메뉴로 선택하여 사용 할 수 있습니다.
5. 스펙트럼 효과를 사용 할 수 있습니다.
6. 스티커를 사용 할 수 있습니다.
7. 자막(문자효과)를 사용 할 수 있습니다.
8. 전체 화면에 바운스 효과를 3단꼐 까지 줄수 있습니다.
9. 인코딩을 HD부터 4K까지 선택하여 코딩 할 수 있습니다.

## 기능별 세부사항
### 스펙트럼 효과
1. 스펙트럼 크기를 트랙별로 지정하여 사용 할 수 있습니다.
2. 스펙트럼 위치를 이동 할 수 있습니다.
3. 위치 이동을 할때는 왼쪽 사이드바의 스팩트럼 설정 버튼을 눌러야 제어 할 수 있습니다.
4. 반사효과가 있는 스펙트럼은 반사효과를 온/오프 시킬 수 잇습니다.
5. 공용색상을 사용하는 스펙트럼은 색상을 자유롭게 변경 할 수 있습니다.

### 자막효과
1. 문자효과는 트랙별 3개까지 사용 할 수 있습니다.
2. 문자효과를 제어하기 위해서는 왼쪽 사이드바의 문자설정 버튼을 눌러 설정창을 열어야 합니다.
3. 문자효과는 크기, 색상, 투명도, 위치이동 등의 기능을 사용 할 수 있습니다.

### 스티커
1. 스티커는 일반 이미지 혹은 투명도가 있는 png, 동영상, 투명도가 있는 webm 동영상을 불러 올 수 있습니다.
2. 스티커는 자체 바운스 기능을 가지고 있으나, 메인에서 바운스 모드를 눌러줘야 작동 합니다.
3. 크기와 위치를 바꿀 수 있으며, 왼쪽 사이드바의 스티커 설정 버튼을 눌러야 이동을 시킬 수 있습니다.

### 객체 이동과 배치
1. 스페이스에 올려진 각 객체는 해당 설정버튼이 활성화 된 상태에서만 이동 시킬 수 있습니다.
2. 이는 여러개의 객체들이 혼재되어 선텍을 제대로 할 수 없게하는 것을 방지하기 위하여 독립적으로 활성화 되도록 하였습니다.

### 저장하기
1. 인코딩 전에 미리보기를 해볼 수 있도록 스페이스 상단에 미리보기 버튼을 제공 합니다.
2. 인코딩 버튼을 누르면 버튼이 붉은 색으로 바뀌며, 언제든지 중단 할 수 있습니다.
3. 인코딩이 중단 된 경우 지금까지의 인코딩한 영상을 자동으로 저장해 줍니다.
4. 저장 폴더는 로컬PC의 다운로드 폴더입니다.
